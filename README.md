### Frontend

1. Make sure you have Node.js and npm installed.

1. Go to the folder.
```sh
cd frontend
```
   
1. Set dependencies using the command.
```sh
npm instal
```
   
1. Start the frontend with the command.
```sh
npm run serve
```
